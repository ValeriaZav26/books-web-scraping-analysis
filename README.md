# 📚 Book Market Analysis  
### Web Scraping + Data Analysis Project

## 📌 Project Overview
This project analyzes the book market using data collected through **web scraping** from the website:

http://books.toscrape.com/

The goal of the project is to demonstrate skills in **web scraping, data cleaning, exploratory data analysis, and data visualization**.

---

## 🎯 Project Goal
The analysis explores the relationship between **book prices and ratings** and answers questions such as:

- What is the average price of books?
- What ratings are most common?
- How are book prices distributed?
- Do higher-rated books tend to be more expensive?

---

## 🛠 Tools Used
- Python  
- Requests  
- BeautifulSoup  
- Pandas  
- Matplotlib / Seaborn  
- Google Colab  

---

## 📊 Data
The dataset contains information about approximately **1000 books**, including:

- Book title  
- Book price  
- Book rating  

---

## 🔎 Workflow
1. Collect data using **web scraping**
2. Clean and prepare the dataset
3. Perform **exploratory data analysis (EDA)**
4. Visualize key insights

---

## 📚 Skills Demonstrated
- Web Scraping  
- Data Cleaning  
- Data Analysis with Pandas  
- Data Visualization  
- Analytical Thinking
