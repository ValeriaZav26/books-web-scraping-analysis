# 📚 Book Market Analysis  
### Web Scraping + Data Analysis Project

## 📌 Project Overview
В этом проекте проводится анализ рынка книг на основе данных, собранных с помощью **web scraping** с сайта:
http://books.toscrape.com/

Цель проекта — продемонстрировать навыки:
- web scraping  
- очистки данных (data cleaning)  
- исследовательского анализа данных (EDA)  
- визуализации данных
